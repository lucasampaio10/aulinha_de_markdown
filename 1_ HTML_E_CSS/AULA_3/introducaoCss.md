# INTRODUÇÃO AO CSS

CSS (Cascading style sheets, ou folhas do estilo em cascata) é a linguagem utilizada para definir o estilo visual de documentos HTML, constrolando o layout, cores, fontes, espaçamento e rersponsividade(forma na qual os componentes se comportam em multiplas telas.).

## Por que usar CSS?

O CSS separa o contéudo (HTML) da apresentação. permite deixar páginas mais atrativas e organizadas
, melhorando a experiencia do usuário. 

## Formas de aplicar css: 

Inline: direto no elemento, usando o atributo style.
Interno: bloco </style> dentro do <head>.
Externo: arquivo .css separado (recomendado para projetos reais).

## componentes css
background color: cor do fundo.
color: cor do texto
font-family: tipo da fonte
font-size: tamanho da fonte
margin: margem de espaçamento externo
